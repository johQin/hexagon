# Negotiation



# 谈判策略-------------

# 1 争取价值

## 1.1 准备谈判的5步法

1. 评估你的最佳备选方案。
2. 计算你的底线。
3. 评估对方的最佳备选方案。
4. 计算对方的底线。
5. 评估协议区间。

## 1.2 如何正确报价

1. 是否应该首先报价？
   - “锚”的作用：当对方设好这个锚，它不但会影响你对对方底线的计算，也会影响你的还价
   - 总的来说，是否应该首先报价取决于你掌握多少信息。
2. 如何应对对方先报价？（5个策略）
   - 忽略对方的锚：当对方的价过于极端，”从你的报价来看，恐怕我们对这场谈判的看法差别很大。为了达成共识，我们不妨先讨论一下××问题。“
   - 区分信息和忽悠
   - 避免陷入对方的锚：
     - 就一个锚的讨论越多，这个锚的影响越大
   - 还一个极端的锚”以暴制暴“
   - 给对方时间和台阶，让对方在不失面子的情况下调整其报价
   - 话术：看来我们需要沟通的内容还有很多，从我们的角度来说，更公平的价格应该更接近xx元（我的极端还价），我会解释为什么我们评估得出了这样的价格，但是，如果想要达成协议的话，还需要我们双方共同努力才行
3. 正确报价的四个因素
   - 好的报价必须把整个协议区间包括在内
   - 为你的报价提供理由：我的报价是xxx，理由是xxx，如果不能合理完成句子，你可能就要的过分了。
   - 设一个很高但可行的目标
   - 报价时一定要考虑谈判背景和双方的关系：拿到好的结果并加强关系



## 1.3 你能把对方逼多狠（我能争取多少价值）

1. 你能把对方逼多狠（我能争取多少价值）
   - 谈判前了解所有信息：功课做足，就不敢轻易骗你，就更拿你当回事
   - 明确你关于谈判的所有假设
   - 质疑假设，获取信息（有锚定，有质询）：如果这片地用作商业开发就非常有价值了（锚定，假设）。我想了解一下，这么好的一片地，你们打算怎么开发呢（质询，获取信息）
   - 迂回询问：让问题显得不那么直接和威胁性
   - 用权变协议保护自己，应对不确定性
2. 讨价还价的7大策略
   - 聚焦对方的底线和最佳备选方案：不能过于关注自己的底线和最佳备选方案，而应反之
   - 不要单方让步：让步必须对等
   - 不要害怕沉默：懂得沉默，在对方开口之前说了话，就要付出代价
   - 给你的让步贴上标签：人们倾向于低估甚至忽略他人的让步，必须强调自己的让步，并让对方知道我的让步是有代价的。并明确表达我期待的回应是什么？
   - 明确定义对等：对方的让步要对等。
   - 使用条件让步：”如果你们能提前发货，我就支付更高的价格“
   - 理解让步的缩小对对方的影响：开始阶段让步大，后期让步小，越接近底线（这种也可能被有意利用）



## 1.4 就关系谈判

谈判中永远有两个明确的目标：争取更好的结果和加强与对方的关系。

你给出一个很好的“锚”的报价。你鼓起勇气提出了这个价格，对方怎么反应呢？对方笑逐颜开，立刻接受了你的报价！你的感受会如何？基本上可以肯定，你会感觉非常糟糕

当对方的报价让你非常开心的时候，怎么回应？

- 法一：假装考虑，平静，拖延一下
- 法二：因为只要你接受了对方的报价，哪怕是表现得很不情愿，他们很可能还是会有遗憾，所以我们应该考虑再还一嘴（还价）
- 法三：慷慨回应慷慨，而不能贪婪
- 法四：提高警惕，他们可能有我们不知道的信息



## 1.5 管理满意度

- 谈判的时候把注意力放在目标上；谈判结束以后把关注点转移到底线上。



# 2 创造价值

## 2.1 多议题谈判

**多议题谈判使创造价值成为可能**

案例：

1. 议题1：播出许可费，电视剧收视率与广告收入
2. 议题2：播放次数，电视剧次数与未来收益
3. 议题3：另一部剧的播出许可费

我的目标是最大化价值

- 引入新议题来创造价值
- 帕累托最优（强调双方的价值总和）：改变一个结果让至少一方受益，但是不会让另一方受损
- 权变协议来创造价值



## 2.2 创造价值的4大准备策略

1. 明确我的多个利益点：
   - 利益点藏在议题背后，议题可能只有一个，但利益点可能有多个
   - 谈薪酬的时候，可以照顾一下自己的其他利益点：带薪假，年度体检，股权等
2. 建立评分系统
   - 组织利益点和优先级排序
3. 计算组合提案的底线值
4. 明确对方的利益点：有些议题是我不在乎，但对方却很在乎，交换价值



## 2.3 创造价值的3大执行策略

1. 同时谈多个议题
   - 逐个议题谈会排除互换的可能性
   - 引入多个议题，可以从”零和“到”非零和“
   - 每个议题的重视程度不同，大家可以交换
2. 给出组合提案
3. 利用各种差异来创造价值



## 2.4 谈判后策略

提出协议后协议的4步法：

1. 首先要重申双方在已经签署的协议上的进展。
2. 表达你还是希望最终结果的某些方面能够有所提高，同时一定要指出对方可能也有同感。
3. 指出你已经作出了所有可能的让步，但是你愿意“开开脑洞”，看看有没有办法帮助对方。
4. 明确表明，最重要的不是要找一个新的协议，而是要找一个双方都认可的、比当前协议更好的协议。

# 3 如何用”侦探式谈判“获取信息



## 3.1 7大原则

用侦探思维去发现议题背后的利益点，而不要停留于表象（是什么，诉求）

1. 不要只问是什么，还要问为什么
   - 过于关注人们“要什么”会分散你的注意力，忘了去发现背后的原因。
2. 努力调和利益点，不要在诉求上纠缠
3. 求同存异
4. 把要求当作机会：对方的要求，隐含了对方的利益点。案例：不按时交付就要缴纳罚款，利用权变协议：延期缴纳，提前则奖励
5. 不要把任何问题简单当作”对方的问题“而打发：
   - 案例：家居廊希望价格低一点，但是也希望能够马上发货；齿轮公司则想要高一点的价格和迟一点的发货时间。根据市场价格，双方同意定价1 700万美元，发货期三个月（家居廊延期则有损失）。但齿轮公司对发货期表示出了担忧：“三个月的发货期对我们来说难度有点大，不过我们会尽力，应该可以应付。”，家居廊以为是生产工艺导致交付困难，但实际是物流的原因，但家居廊可以在物流上帮助齿轮公司
6. 不要让谈判以自己的提案被拒作为终结
   - 谈判绝不应该以对方拒绝作为结束，而是要么以同意结束，要么以得到“为什么不”的解释结束。
7. 推销和谈判不一样的



## 3.2 从口风紧的谈判者获取信息

**5大策略**

1. 建立信任，分享信息
   - 用对方的语言沟通
   - 加强各种联系的纽带（分享共同的朋友，在共同的社区生活工作）
   - 在谈判之外建立信任：建立信任最好的时机，是对方不觉得你的合作和善意具有目的性时。
2. 提问：在惊讶和有疑虑时
3. 提供一些自己的信息：互惠原则，在任何情况下，都不能告诉对方我们的底线，至少谈判初期绝不可以，分享议题的优先级
4. 同时就多个议题谈判：对方强调哪个议题，说明哪个议题就会更重要
5. 同时给出多个提案：给出两个对自己价值一样的，但内容侧重点稍有区别的提案，这样就可以了解对方的偏好了
